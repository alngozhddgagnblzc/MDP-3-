## ⚙️ 우리의 작품
+ ### **스마트 냉장고 (Smart Refrigerator)**
+ 나의 냉장고 안에 식재료를 파악해 앱으로 간단하게 알려주고, 인공지능을 이용해 현재 가지고있는 식재료를 기반으로 레시피를 추천해주는 기능을 가진 냉장고와 앱
